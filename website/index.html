Atalhos do teclado

  • CTRL + SHIFT + P: wrap with abbreviation
  • SHIFT + TAB: tira uma identação selecionada
  • SHIFT + ALT + F: formata o documento
  • ALT + click: cursor em vários locais
  • CTRL + space: mostra os arquivos (sugestões)
  • CTRL + /: comenta a linha
  • CTRL + D: seleciona a palavra
  • CTRL + SHIFT + L: seleciona todas as palavras iguais
  • CTRL + SHIFT + K: deleta a linha
  • CTRL + Click: cria um arquivo com o link que você digitou

Estilos CSS

Pode ser inline, interno e externo

  • *{} - configuração global CSS
  • @charset "UTF-8"; - regra
  • @import url - importar fonte
  • @font-face - importar fonte otf-ttf
    • font-family: 'nome da fonte;
    • src: ulr('love larry.otf') format ('opentype'), ulr('love larry.ttf') format('truetype');
  • Seletores específicos
    • id# - único: h1#principal (h1 opcional)
    • class. - vários elementos: h1.principal (h1 não põe - prof) (pode ter mais de uma classe)
    • pseudo-classe: (estado do elemento) - h1:hover/active/visited
    • pseudo-elemento:: - (adiciona um elemento) h1::after/before - mexe no conteúdo periférico do elemento
  • (children) tag 1 > tag 2 (só aplica se a tag 2 estiver dentro da tag 1 no html) - aplica para a tag 2

    • Textos

  • font-family: muda a fonte
    • sans-serif, serif, monospace, cursive
  • text-align: justify/center/right/left
  • text-indent - identação do texto
  • font-size
  • font-weight - peso da fonte (bold)
  • font-style - estilo da fonte (itálico)
  • text-decoration - estilo da fonte (sublinhado)
  • font - shorthand (resume os comandos)
    • font-style > font-weight > font-size > font-family
    • exemplo::: font: italic bolder 3em 'Work Sans', sans-serif;
