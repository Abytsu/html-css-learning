Atalhos do teclado

  • CTRL + SHIFT + P: wrap with abbreviation
  • SHIFT + TAB: tira uma identação selecionada
  • SHIFT + ALT + F: formata o documento
  • ALT + click: cursor em vários locais
  • CTRL + space: mostra os arquivos (sugestões)
  • CTRL + /: comenta a linha
  • CTRL + D: seleciona a palavra
  • CTRL + SHIFT + L: seleciona todas as palavras iguais
  • CTRL + SHIFT + K: deleta a linha
  • CTRL + Click: cria um arquivo com o link que você digitou

Tags

Texto e formatação

  • h1 - título nível 1
  • p - parágrafo
  • address - endereço
  • q - citação
  • blockquote - citação mais longa
    • cite = "Leonardo daVinci"
  • abbr - abreviação
  • bdo - texto invertido
  • style - cria um estilo CSS para o documento
  • sup - sobrescrito
  • sub - subscrito
  • code - linhas de código
  • pre - deixa a formatação do VsCode

    • Marcações

  • strong - palavra forte (b)
  • em - palavra ênfase (i)
  • mark - marca texto
  • small/big
  • del - texto deletado
  • ins - texto inserido (u)

    • Listas

  • ul - lista não ordednada
    • type = "disc/circle/square"
  • ol - lista ordenada
    • type = "1, A, a, I, i"
    • start = "1,2,3,4,5,6..."
  • li = list item (não precisa fechar sempre)
  • dt - lista termo
  • dd - lista descrição

Elementos

  • hr - linha horizontal
  • br - quebra de linha
  • link:favicon - altera o favicon do website
  • link:css - linka o style.css
  • a - âncora para hypertexto (links)
    • href = "trajeto"
    • target = "_blank/_self"
    • rel = "external/prev/next/nofollow/dofollow
    • download = "index.pdf"
    • type = "application/pdf"

Mídia

  • img - imagem
    • width
    • height
  • picture - permite imagem com vários source
  • source - source para pictures/audio/video...
    • media:type
  • audio - áudio para o site (src)
    • autoplay
    • controls
    • loop
  • video - vídeo para o site (src)
    • width
    • controls
    • poster - thumbnail do vídeo

Estilos CSS

Pode ser inline, interno e externo

  • background-color: muda a cor de fundo
  • font-family: muda a fonte
  • text-align: justify
  • font-size
  • @charset "UTF-8"; - regra

Vale lembrar (caracteres especiais)

  • &lt; e &gt; → < >
  • &#x1F601 → 😁 (emojipedia.org)
  • "£ $ ¥ ¢ →"
