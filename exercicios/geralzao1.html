Atalhos do teclado

  • CTRL + SHIFT + P: wrap with abbreviation
  • SHIFT + TAB: tira uma identação selecionada
  • SHIFT + ALT + F: formata o documento
  • ALT + click: cursor em vários locais
  • CTRL + space: mostra os arquivos (sugestoes)
  • CTRL + /: comenta a linha
  • CTRL + D: seleciona a palavra
  • CTRL + SHIFT + L: seleciona todas as palavras iguais
  • CTRL + SHIFT + K: deleta a linha
  • CTRL + Click: cria um arquivo com o link que você digitou

Marcas

    Texto e formatação

  • <h1> - título nível 1
  • <p> - parágrafo
  • ol type="" start="": lista ordenada
  • ul type="disc/circle/square">: lista não ordenada
  • li: list item (não precisa sempre fechar)
  • dt: lista termo
  • dd: lista descrição
  • <address> - endereço
  • <q>: citação
  • <blockquote cite=""> - citação mais longa
  • <abbr> - Abreviações
  • <bdo> - texto invertido
  • <strong> → <b> - palavra forte
  • <em> → <i> - palavra ênfase
  • <mark> - marcar texto
  • <style> - cria um estilo CSS para o documento
  • <small> (based) e <big> (obsoleto)
  • <del> e <ins> - texto deletado e texto adicionado → <u>
  • <sup> - sobrescrito123
  • <sub> - subscrito123
  • <code> - linhas de código
  • <pre> - deixa a formatação

    • Elementos

  • <hr> - linha horizontal
  • <br> - quebra de linha
  • <link:favicon> - altera o favicon
  • link:css - linka o CSS
  • a href="" target="_blank/_self" rel="external/prev/next/nofollow"/dofollow download="" type="application/pdf">: âncora para hypertexto (links)

    • Midia

  • <img> - imagens para o site
  • picture - permite que img tenha vários source
  • source:media:type - source das imagens com diferente tamanho dependendo do monitor
  • audio:autoplay:controls:loop - coloca audio no site, pode ser usado com source (src="")
  • video:width:controls:poster - video, pode ser usado source (src)
